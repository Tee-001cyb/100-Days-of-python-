# Basic Authentication System

This is a simple Python authentication system that allows users to register and log in using a username and password.

The program presents a menu where users can choose to register, log in, or exit the application. During registration, it checks whether the chosen username already exists before storing the username and password. During login, it verifies the entered credentials and grants access only if they match the stored information.

All user data is stored in a Python dictionary while the program is running, making it a simple demonstration of authentication logic without using a database.

## Features

* Register a new user with a unique username.
* Prevent duplicate usernames during registration.
* Log in using registered credentials.
* Validate usernames and passwords.
* Menu-driven interface with options to register, log in, or exit.
* Stores user information in a Python dictionary during runtime.

## Purpose

I built this project to practice working with Python dictionaries, functions, conditional statements, loops, and user input. It also helped me understand the basic logic behind user authentication systems, including user registration, credential validation, and simple data storage.
