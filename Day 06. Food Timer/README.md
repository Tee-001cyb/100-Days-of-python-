# Food Timer

This is a simple Python countdown timer designed to help users keep track of cooking times for different food items.

The program displays a list of predefined foods along with their cooking times in seconds. Users can select a food item to start a countdown timer. If the selected food is not available in the list, the program allows the user to add a new food item and specify its cooking time before starting the timer.

The countdown is displayed in **HH:MM:SS** format and updates every second until the timer reaches zero, after which the program notifies the user that the food is ready.

## Features

* Displays a list of predefined food items and their cooking times.
* Starts a countdown timer for the selected food.
* Shows the remaining time in **hours, minutes, and seconds (HH:MM:SS)** format.
* Allows users to add new food items and their cooking times.
* Updates the timer in real time until the countdown is complete.
* Notifies the user when the cooking time has ended.

## Purpose

I built this project to strengthen my understanding of Python's `time` module, particularly the `sleep()` function for creating real-time delays. It also gave me hands-on experience with dictionaries, loops, functions, user input, arithmetic operations for time conversion, string formatting, and building a practical command-line application.
