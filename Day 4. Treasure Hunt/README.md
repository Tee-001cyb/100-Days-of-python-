# Treasure Hunt Game

This is a simple text-based adventure game built with Python. The player must make a series of choices to navigate through different scenarios in search of a hidden treasure.

The game begins at a crossroads where the player's decisions determine the outcome of the adventure. Choosing the correct path allows the player to progress, while incorrect choices result in a game over. After each game, the player has the option to restart and try again until they successfully find the treasure.

## Features

* Interactive text-based gameplay.
* Multiple decision points that affect the outcome of the game.
* Win and lose scenarios based on the player's choices.
* Option to restart the game after winning or losing.
* User-friendly menu driven by keyboard input.

## Purpose

I built this project to practice working with Python functions, conditional statements (`if`, `elif`, and `else`), loops, user input, and string methods. It also helped me improve my understanding of program flow, decision-making logic, and creating interactive command-line applications.
