# Rock, Paper, Scissors Game

This is a simple command-line implementation of the classic Rock, Paper, Scissors game built with Python. The player competes against the computer, which randomly selects one of the three available options each round.

The program compares the player's choice with the computer's choice, determines the winner based on the game rules, and displays the result. After each round, the player can choose to continue playing or exit the game.

## Features

* Play against a computer opponent.
* Computer makes a random choice each round.
* Determines whether the player wins, loses, or ties.
* Displays both the player's and computer's choices.
* Allows the player to continue playing multiple rounds or exit the game.
* Uses a dictionary to simplify the game logic for determining the winner.
