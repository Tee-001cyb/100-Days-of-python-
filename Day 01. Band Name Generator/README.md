Brand Name Generator

This is a simple Python program that generates a unique brand name based on user input.

The program prompts the user to enter their favorite country and their pet's name. It then combines both inputs to generate a creative brand name and displays the result.

Features

* Accepts user input for a favorite country.
* Accepts user input for a pet's name.
* Combines both strings to generate a brand name.
* Displays the generated brand name to the user.

Purpose

I built this project to practice working with user input, string concatenation, dictionaries, and functions in Python. It helped reinforce how to collect user input, manipulate strings, organize data, and create reusable functions.
