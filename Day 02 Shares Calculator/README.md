# Payment Calculator

This is a simple Python payment calculator that helps split a restaurant bill among a group of people.

The program first prompts the user to enter the total bill amount. It then asks whether they would like to leave a tip. If the user chooses to tip, they can enter the desired tip percentage, which is added to the original bill before calculating the final total.

After determining the total amount (with or without a tip), the program asks for the number of people sharing the bill. It then calculates and displays how much each person should pay.

## Features

* Accepts the total bill amount.
* Allows users to add an optional tip by specifying a percentage.
* Calculates the final bill, including the tip (if applicable).
* Splits the total bill equally among the specified number of people.
* Displays the amount each person is expected to pay.

## Purpose

I built this project to strengthen my understanding of Python's arithmetic operations and assignment operators while practicing user input, conditional statements, and basic calculations.
